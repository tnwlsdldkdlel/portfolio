# 포트폴리오 개발 워크플랜

## 프로젝트 개요
성능 중심의 기술 서사를 시각화하는 웹 포트폴리오 구축
- 목표: Lighthouse 점수 모든 항목 100점
- 디자인: Strict Black & White (B&W)
- 프레임워크: Next.js 16 (App Router) + Tailwind CSS v4

## 기술적 고려사항

### 성능 목표
- Lighthouse Performance: 100점
- Lighthouse Accessibility: 100점
- Lighthouse Best Practices: 100점
- Lighthouse SEO: 100점

### 디자인 원칙
- Strict Black & White만 사용
- Invert 효과로 강조
- 굵은 타이포그래피 (Montserrat)
- 미니멀한 레이아웃

### 데이터 관리
- 정적 데이터는 JSON 파일 또는 TypeScript 상수로 관리
- Velog API는 서버 컴포넌트에서 호출 (ISR 활용 고려)
